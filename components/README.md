# Папка `components/`

Сюда складываем **переиспользуемые** части интерфейса:

- то, что повторяется на нескольких страницах (шапка, футер, кнопка);
- то, что логично вынести из `page.tsx`, чтобы файл страницы оставался коротким.

**Именование:** `PascalCase.tsx` для компонентов (как `SiteHeader.tsx`).

**Импорт:** из кода приложения удобно писать `@/components/...` благодаря алиасу в `tsconfig.json`.

На этапе 2 добавлен **`CarCard`** для сетки каталога; дальше — формы и прочие блоки.
