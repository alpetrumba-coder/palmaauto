# Папка `app/` (Next.js App Router)

- **`layout.tsx`** уровня `app/` — общий каркас: `<html>`, `<body>`, подключение `globals.css`.
- **`(public)/`** — всё, что видит посетитель: главная `/`, в будущем каталог и вход.
- **`(internal)/`** — зона сотрудников; сейчас есть страница **`/staff`**.

Скобки в имени папки — **группа маршрутов**: имя группы **не** входит в URL, но позволяет повесить **свой** `layout.tsx` на часть сайта.
