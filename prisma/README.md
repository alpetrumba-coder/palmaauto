# Папка `prisma/`

- **`schema.prisma`** — единый источник правды по таблицам и связям. После изменений выполняйте миграцию или `db push` (см. корневой `README.md`).
- **`migrations/`** — пошаговая история изменений БД для продакшена и команды.

Первая миграция `20250409120000_init` создаёт таблицу `users` и enum ролей — заготовка этапа 1.
