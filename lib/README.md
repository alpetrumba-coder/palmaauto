# Папка `lib/`

Здесь **не UI**, а код, который:

- подключается к базе (`prisma.ts` — один экземпляр клиента Prisma на процесс в dev);
- запросы каталога (`cars.ts`), формат цены (`formatPrice.ts`);
- проверка адресов фото каталога (`carImageSrc.ts` — локальный путь `/cars/...` или `http`/`https` URL).

Инструкции по папке с файлами фото: **`public/cars/README.txt`**.

Импорт: `@/lib/...`.
